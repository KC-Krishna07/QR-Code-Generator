# QR Studio Pro

Enterprise-style QR Code Generator with Local Asset Management.

## 🚀 Features
- High-fidelity QR code generation
- LocalStorage-based asset history
- Instant re-generation from history
- PNG export
- Clean enterprise UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- QRCode.js Library

## 📦 How to Run
1. Clone the repository
2. Open `index.html` in browser

## 🌐 Future Improvements
- SVG Export
- QR Custom Colors
- Logo Embedding
- Cloud Sync
